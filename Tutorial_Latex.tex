\begin{abstract}
    LaTeX est un langage informatique qui permet de formatter et de générer des documents
    scientifiques.
    LaTeX est notamment utilisé par les mathématiciens pour la qualité du rendu et de la
    génération de formules mathématique.

    Ce document s'adresse à la fois aux néophytes qui souhaitent apprendre à utiliser \LaTeX\
    qu'aux utilisateurs chevronnés qui veulent trouver rapidement une commande spécifique.
    La version courante de ce document est disponible sur GitHub à l'adresse
    suivante:
    \begin{center}
        \url{https://github.com/Falkor/latex_tutorial}.
    \end{center}
\end{abstract}

\section{Liens utiles}
\label{sec:links}

Outre la bibliographie qui fournit notamment les livres de références sur \LaTeX\
\cite{Lamport,Latex-oReilly}, il existe un certain nombre de documents
qui peuvent vous être utiles en complément de ce tutorial:
\begin{itemize}
  \item FAQ (Frequently Asked Questions) \LaTeX:\\
    \url{http://www.grappa.univ-lille3.fr/FAQ-LaTeX/}
    Ce site devrait répondre à toutes vos interrogations sur \LaTeX\
    et la façon de réaliser telle ou telle chose.
  \item Le site de référence  pour Latex2e:\\
    \url{http://cs.wlu.edu/~necaise/refs/latex2e/Contents.html}
  \item Le site \url{http://www-h.eng.cam.ac.uk/help/tpl/textprocessing/} référence
    bon nombres de documents sur \LaTeX.
  \item J'aime beaucoup le site \url{http://www.commentcamarche.net/} qui
    contient notamment un tutorial de Thomas Nemeth sur Latex \cite{NemethLatex}.
  \item Linda CHAN-SUN a écrit un tutorial \cite{ChansunLatex} pour les étudiants
    de supinfo.
  \item Le "vieux" tutorial de Frédéric Geraerds \cite{GeraerdsLatex} pourra
    également vous être utile.
  \item Un tutorial en anglais avec quelques exercices: \\
    \url{http://www.cs.cornell.edu/Info/Misc/LaTeX-Tutorial/LaTeX-Home.html}
  \item "Installer \LaTeX\ sous Windows"\\
    \url{http://mathro.fpms.ac.be/~glineur/LaTeX/index.fr.html}
\end{itemize}
